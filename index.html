Shopping Cart

This project is a simple shopping cart web application built using HTML, CSS, and JavaScript. It allows users to view a list of products displayed in a single horizontal row, with each product showing an image, name, price, and an "Add to Cart" button. Users can add items to the cart, increase or decrease the quantity, and view the updated total order value in real time. The application uses JavaScript for managing cart operations and dynamically updating the DOM, while CSS (using Flexbox) ensures a clean, responsive layout. This project demonstrates fundamental front-end development skills, including DOM manipulation, event handling, and layout design using modern CSS techniques.

My Bank

This project, titled "My Bank", is a simple front-end user authentication system built using HTML, CSS, and JavaScript. It allows users to register by entering their name, email, and password, after which they are added to an in-memory array with a default account balance of ₹1000. Once registered, users can log in using their email and password. If the credentials match, the user is welcomed on a home page with a logout option; otherwise, an "Access Denied" message is shown. The application also displays a list of all registered users alongside the login form, showing their name, email, password, and balance (for demonstration purposes). The user interface dynamically switches between login and registration forms using JavaScript. This is a client-side only application with no database or backend, meaning all data is temporary and resets on page reload. It is designed as a learning project to demonstrate basic user authentication logic and UI handling using JavaScript.
